# DocForge ML System Design

Проектная документация по разработке сервиса DocForge в рамках курса ML System Design and My First Data Project.

## О проекте

DocForge — интеллектуальная платформа подготовки, валидации и загрузки технической документации в корпоративные базы знаний и RAG-системы

Платформа позволяет автоматизировать процесс обработки технических документов, снизить нагрузку на команды Data Science и передать задачи наполнения базы знаний предметным экспертам.

Основные возможности системы:

* загрузка технических документов;
* извлечение структуры документа;
* проверка и редактирование результатов обработки;
* семантическое разбиение документов на чанки;
* подготовка данных для векторных баз знаний;
* загрузка данных в Knowledge Base;
* поддержка процессов сопровождения RAG-систем

## Цель проекта

Создать платформу, которая позволит предметным экспертам самостоятельно готовить документы для корпоративных баз знаний без необходимости взаимодействовать с ML-инфраструктурой, векторными БД и внутренними пайплайнами обработки данных
